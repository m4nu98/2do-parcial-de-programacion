 ## Características

*   **Formulario de ingreso de datos:** Permite a los usuarios agregar nuevos vehículos a la base de datos, especificando la siguiente información:
    *   Marca
    *   Año
    *   Modelo
    *   Motor
    *   Color
*   **Base de datos:** La información de los vehículos se almacena en una base de datos MongoDB.
*   **Servidor web:** La aplicación utiliza un servidor Express.js para manejar las solicitudes del cliente y la interacción con la base de datos.

## Cómo usar

1.  Abre el archivo `public/index.html` en tu navegador web.
2.  Completa el formulario con la información del vehículo.
3.  Haz clic en el botón "Guardar".
4.  La información del vehículo se guardará en la base de datos.
